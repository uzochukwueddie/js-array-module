# js-array-module
A simple module that provides functions to manipulate arrays

## HOW TO INSTALL
# npm install js-array-module --save

```javascript
##  HOW TO USE
var array = require('js-array-module');

## METHODS
1. SumArray
2. AverageArray
3. MaxArray
4. MinArray
5. UniqArray
6. UniqArraySort
7. UniqStringArray
```
